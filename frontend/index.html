Privatni server

Šalje dekodirane pakete na vaš server paralelno sa SondeHub-om.

UDP: fire-and-forget, brže. TCP: pouzdanije, ali sporije.
JSON šalje sva polja iz paketa. CSV šalje: callsign, time, lat, lon, alt, speed, heading, snr, battery, temp, sats.

Postavke privatnog servera se spremaju zajedno sa stanicom (Spremi u sidebaru).
